Bienvenue



    Bienvenue sur le jeu seman-game ! Voici une FAQ pour répondre aux questions sur l'utilisation du jeu.

    Que faut-il faire ?

    Le jeu seman-game vous challenge à essayer de trouver le nom d'un article Wikipédia en s'aidant simplement de la sémantique des mots.

    Comment est-ce que je démarre le jeu ou que je génère un article ?

    Rien de plus simple, vous avez plus haut un bouton appelé Nouvel article. Deux options s'offrent à vous en dessous de ce bouton, vous pouvez au choix choisir un article complètement aléatoire depuis l'entièreté du Wikipédia français ou de choisir un article parmi une liste d'article pré-définis qu'il est possible de fournir manuellement.
    La première option, l'option aléatoire, prend généralement plus de temps à se générer, il est en effet nécessaire d'écremer tous les articles que nous renvoient Wikipédia.
    La seconde option, l'option d'un article parmi une liste, est plus rapide mais vous risquez de tomber sur un article que vous avez déjà recherché.

    Comment gagner une partie ?

    Il vous suffit simplement de retrouver le titre exact et entier de l'article.

    Pourquoi y a-t-il des points avec des espaces blancs sur la page quand je la génère ?

    Ces points correspondent à des mots cachés par le jeu. En effet, les mots sont tout d'abord cachés afin de ne pas revelé la page directement. Chaque lettre est remplacé par un point afin de représenter la vraie taille du mot caché et donner une idée de ce qui est recherché. Les caractères et les espaces sont bien évidemment révélés.

    Comment "trouver" un mot ?

    Pour trouver un mot, le rendre clair dans l'aire de jeu, il suffit de le taper dans le champ textuel tout en haut de la page. Dès lors, plusieurs évènement vont se produire:

    • Le mot tapé correspond exactement à un ou plusieurs mot de l'article. Le ou les mots correspondants vont s'afficher en clair avec un fond vert pendant quelques secondes.
    • Le mot tapé a une sémantique proche de un ou plusieurs mots de l'article. Plus la sémantique est proche, plus le mot s'affichera dans une couleur claire, sur un fond noir, de rouge à jaune.
    • Le mot tapé ne correpond à aucun mot ou ce n'est pas un mot existant dans le dictionnaire, dans ce cas rien ne se passe.

    La sémantique ? Qu'est-ce que c'est ?

    La sémantique est l'étude de la signification des mots et de la façon dont ils sont reliés les uns aux autres dans une langue. Le calcul de la similarité entre deux mots consiste à mesurer la ressemblance ou la proximité sémantique entre eux.
    Nous utilisons un outil effectuant ce calcul de similarité entre mots en utilisant un modèle de traitement du langage naturel tels que le modèle de mots vectoriels entraîné avec Wikipédia. Les modèles vectoriels représentent les mots sous forme de vecteurs numériques, ce qui permet de mesurer la similarité en utilisant des métriques mathématiques telles que la distance cosinus.
    Deux mots proches sémantiquement ont donc un sens proche dans une phrase donnée. Par exemple les mots "haut" et "grand" qui sont tous les deux des adjectifs pouvant avoir un sens commun dans certains contexte pour décrire un bâtiment par exemple. Ou deux comme "France" et "Belgique" qui sont de proches voisins physiquement aussi.
    Néanmoins, il peut arriver que deux mots opposés sémantiquement aient une similarité proche avec notre modèle.

    Puis-je jouer contre quelqu'un sur le même article ?

    C'est une très bonne question, seman-game propose de générer un code correspondant à l'article sur lequel tu es en train de jouer. En partageant le code à d'autres personnes il est possible de s'affronter, ou de chercher, ensemble sur le même article.
    Pour faire cela il suffit de cliquer sur le bouton Générer code. Un code est ensuite généré que vous pouvez copier à l'aide du bouton Copier.
    Vous pouvez ensuite partager le code à qui vous voulez. La personne ayant le code peut donc coller le code dans le champ de texte Code de l'article, cliquer sur Entrer le code et voilà !
    ⚠️ Ceci ne fonctionne que lorsque vous jouez sur le même serveur. Il n'est pas possible de rentrer le code et s'attendre à avoir le même article sur deux serveurs différents.

    Je veux ajouter des mots dans la liste pour tout le monde, comment faire ?

    Il est possible d'ajouter des articles dans la liste utilisée pour la génération de l'article rapide. Pour cela rendez vous sur la page de sélection des articles en cliquant sur le bouton correspondant sur la page princpale.
    Vous pouvez ainsi générer un nouvel article aléatoire qui sera affiché en clair par la suite en cliquant sur Nouvel article. De là vous pouvez choisir ou non de l'ajouter à la base de données des articles en cliquant sur Oui ou Non.
    Nouveau ! Vous pouvez ajouter manuellement un article à la liste sans passer par la génération. Attention le jeu va s'assurer qu'il s'agit d'un vrai article. Attention aussi aux homonymes, par exemple Python va renvoyer une page contenant tous les homonymes de Python mais Python (langage) va retourner le langage de programmation.
